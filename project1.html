GUI Application for Database Manipulation

Overview

This project is a desktop application designed to simplify database management. It provides a user-friendly graphical interface (GUI) for performing complex database operations without writing raw SQL queries. The application connects to a MySQL database and allows for intuitive data manipulation, viewing, and editing.

Key Features

  • User-Friendly Interface: A clean and intuitive GUI that makes database interactions straightforward.
  • CRUD Operations: Full support for Create, Read, Update, and Delete operations on database records through simple forms and buttons.
  • Data Visualization: Presents database tables in a clear, readable format.

Technologies Used

Backend & GUI: Python (with a GUI library like Tkinter or PyQt)
Database: MySQL
