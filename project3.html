3D Interactive Portfolio Website

Overview

This ongoing project aims to redefine the concept of a personal portfolio. Instead of a traditional webpage, it's an interactive 3D world where visitors can explore my projects in a game-like environment. This portfolio showcases my skills in both full-stack web development and 3D graphics, blending them into a unique user experience.

Key Features

  • Interactive 3D Scene: A navigable 3D space built with Three.js.
  • Dynamic Content: Project information is loaded dynamically from a backend database.
  • Responsive Design: The 3D experience is designed to work smoothly on both desktop and mobile browsers.

Technologies Used

Frontend: React, Three.js
Backend: Node.js, Express.js
Database: MongoDB
(This constitutes the MERN stack)
