Brochure Website for Western Opticals

Overview

As a project, I designed and developed a WebGL Game. The primary goal was to create a clean, modern online interaction that showcases 3D models and web interactions. The app is fully responsive and optimized for a great user experience.

Key Features

  • 3D: A 3D layout that reflects the professionalism of the business.
  • Responsive Layout: The site adapts perfectly to desktops, tablets, and mobile phones.
  • 3D Interactive: Fully interactive 3D app.
  • Made in Spline: Made entirely in Spline.

Technologies Used

Frontend: HTML5, CSS3, JavaScript, Spline
